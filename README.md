# How to Run

## Prerequisites

- Python 3.8+
- Node.js 14+
- Docker (for Docker setup)


## Using Script Files in root folder

**Windows:**

```bash
start.bat
```
**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```


## Using Docker

```bash
docker-compose up --build
```

Access the application at:
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
